# MenuHub Project Context

## Proje amacı
Restoranların text tabanlı menü ve fiyat bilgisini yayınlayan bir sistem.
Kullanıcı restoran ve menüleri görüntüler.
Admin restoranları ve menü ürünlerini yönetir.

## Teknoloji
- Backend: Spring Boot
- Frontend: Next.js / React
- Database: PostgreSQL
- Infra: Docker
- IDE: IntelliJ IDEA

## Repo
- GitHub: `https://github.com/berkayarslan/menuhub`

## Çalıştırma
### Backend
IntelliJ üzerinden yeşil run tuşu ile çalıştırılıyor.
Terminal üzerinden Maven bazen local repo sorunu çıkardı; IntelliJ run daha stabil çalıştı.

### Frontend
```bash
cd frontend
npm install
npm run dev
